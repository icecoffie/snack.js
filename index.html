Documentation (FAQ Style)

Clean, readable, and comprehensive

snack.js adalah *utility library* modern dan ringan yang dirancang untuk *frontend developer* yang menghargai kesederhanaan dan kinerja. Dengan *zero dependencies* dan dukungan penuh ES module, Anda dapat mulai menggunakan utilitas canggih dalam hitungan detik.

üéØ Core Philosophy

  • Small, focused utilities
  • Tree-shakeable by default
  • No external dependencies
  • TypeScript support included

üöÄ Quick Start

Install dan gunakan dalam waktu kurang dari 30 detik:

npm i snack.js

debounce(fn, delay)

Membuat fungsi *debounced* yang menunda pemanggilan fungsi sampai setelah jeda waktu tertentu berlalu.

const dSearch = debounce(fn, 300);

Search Input with Debounce

import { debounce } from 'snack.js';
const input = document.querySelector('#search');
input.addEventListener('input',
  debounce((e) => console.log(e.target.value), 300)
);
